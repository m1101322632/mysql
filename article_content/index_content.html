MySQL 5.6 参考手册

包含 MySQL Cluster NDB 7.3-7.4 参考指南

摘要

本手册分别记录了MySql5.6到5.6.32版本、基于版本7.3和7.4发布的 NDB 引擎的 5.6.30-ndb-7.3.14和5.6.30-ndb-7.4.12版本的参考内容。

MySQL 5.6 特点.  这个手册中描述的特点并不被MySql5.6版本下的所有子本版包含。这些特点可能没有被包含在你使用的MySql5.6许可版本中。 如果你对你所使用的mysql5.6版本中的特点有任何疑问，请参考你的5.6版的许可协议或者联系oracle销售代表。

请在 MySQL 5.6 Release Notes中查看每个发不版本的详细说明.

对于法律信息，请参考 法律信息章节.

使用MySql的过程中想要求助的话，请访问 MySQL Forums 或 MySQL Mailing Lists, 在那里你可以和其他mysql用户讨论你遇到的问题。

对于MySql产品的其他文档，包括其转换成其他语言的参考手册和其对应的各种格式（例如HTML、PDF）文件的下载信息，可以在 MySQL Documentation Library中进行查看。

许可信息--MySQL 5.6.  这个产品可能包含第三方软件，使用下面的许可信息。 如果你使用了商业版 的MySQL 5.6, 请参考这个文档了解许可信息, 里面包括了商业版中引用到的第三 方软件的许可信息。 如果你使用的是社区版 的MySQL 5.6, 请参考 这个文档 来了解包含社区版引用到的第三方软件在内的许可信息。

许可信息--MySQL 集群.  这个软件可能在许可使用的前提下使用了第三方软件。如果你使用了 商业版的MySQL Cluster NDB 7.3 or NDB 7.4, 请参考 这个文档 来了解包含商业版引用到的第三方软件在内的许可信息。如果你使用了 社区版 的 MySQL Cluster NDB 7.3 or NDB 7.4, 请参考这个文件 来了解包含社区版引用到的第三方软件在内的许可信息。
